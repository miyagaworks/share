## 第0部：エグゼクティブサマリー

### 目的
名刺印刷会社・Web制作会社・広告代理店の経営者に対し、
「Share」デジタル名刺のホワイトラベルパートナーとして参画するメリットを伝え、
**資料請求または無料相談の予約**というアクションを獲得する。

### ターゲット
- **一次:** 名刺印刷会社の経営者（45〜65歳、従業員5〜30名）
- **二次:** Web制作会社・広告代理店の経営層

### コンバージョンゴール
- **主目標:** 資料請求フォーム送信（CVR目標: 3〜5%）
- **副目標:** 無料相談予約（CVR目標: 1〜2%）

### LP全体のコアメッセージ
> 「紙の名刺をやめる必要はない。でも、"紙だけ"の時代は終わった。
> 　御社のブランドで、デジタル名刺事業を今日から始められます。」

### 感情の設計サマリー
```
ファーストビュー    → 「これ、うちの話だ」（認知・関心）
問題提起          → 「やっぱりこのままじゃまずい」（不安の顕在化）
共感              → 「わかってくれる人がいた」（安心・信頼）
業界変化          → 「もう動き出してる会社があるのか」（焦り→希望）
解決策            → 「これならできるかも」（期待）
ビジネスモデル     → 「具体的に儲かるイメージが湧く」（確信）
差別化            → 「他にはない強みがある」（優越感）
導入の簡単さ      → 「ITに弱くても本当に大丈夫そうだ」（安心）
社会的証明        → 「もう始めてる会社がある」（後押し）
オファー          → 「この条件なら試す価値がある」（興奮）
限定性            → 「今動かないと枠がなくなる」（緊急性）
CTA              → 「まずは話だけでも聞いてみよう」（決断）
追伸              → 「この人と一緒にやりたい」（共感・信頼の最終確認）
```

## 第1部：LPの戦略設計

### 1-1. コピーライティングフレームワーク統合設計

本LPでは3つのフレームワークを以下のように統合する。

| セクション | 新PASONA | QUEST | SB7（ストーリーブランド） |
|-----------|----------|-------|------------------------|
| ファーストビュー | ― | Qualify（選別） | 主人公の紹介 |
| 問題提起 | Problem（問題） | Understand（理解） | 主人公の課題 |
| 共感 | Affinity（親近感） | Understand（理解） | ガイドの登場 |
| 業界変化 | Problem深掘り | Educate（教育） | 外的要因の提示 |
| 解決策 | Solution（解決策） | Educate（教育） | ガイドが計画を示す |
| ビジネスモデル | Solution具体化 | Stimulate（刺激） | 成功のビジョン |
| 差別化 | Solution強化 | Stimulate（刺激） | 他の選択肢との比較 |
| 導入の簡単さ | ― | Stimulate（刺激） | 行動を促す |
| 社会的証明 | ― | Stimulate（刺激） | 変身の証拠 |
| オファー | Offer（提案） | Transition（転換） | 行動喚起 |
| 限定性 | Narrowing（絞込） | Transition（転換） | 失敗の回避 |
| CTA | Action（行動） | Transition（転換） | クライマックス |
| 追伸 | ― | ― | エピローグ |

**ストーリーブランド（SB7）の骨格:**
- **主人公:** 名刺印刷会社の社長（紙の名刺一筋30年、でも将来が不安）
- **課題:** 市場縮小・単価下落・顧客の流出
- **ガイド:** Share（印刷業界を理解し、技術力を持つパートナー）
- **計画:** ホワイトラベルで自社ブランドのデジタル名刺事業を立ち上げる
- **行動:** 資料請求 → 無料相談 → トライアル開始
- **回避すべき失敗:** 何もせず市場縮小に飲まれる
- **成功:** 紙＋デジタルの両輪で、売上と顧客基盤を拡大

### セクション2：問題提起（Pain / Problem）

**目的:** 「このままではまずい」を数字とストーリーで実感させる
**感情:** 不安 → 危機感の顕在化
**心理トリガー:** 損失回避

#### 見出しコピー案

| # | コピー案 |
|---|---------|
| 1 | **名刺の注文、5年前と比べてどうですか？** |
| 2 | **「去年より減った」。その感覚、正しいです。** |
| 3 | **あなたが感じている不安は、データが裏付けています。** |

**推奨: 案1（問いかけ型で自分事化しやすい）**

#### 本文の方向性

**数字で突きつけるパート:**
- 名刺印刷市場の縮小データ（ペーパーレス化、リモートワーク普及）
- ネット印刷の台頭による単価下落率
- 「名刺交換しない」ビジネスパーソンの増加率
- デジタル名刺市場の成長率（年6.8%、2032年に672億円規模）

**チェックリスト「こんなこと、感じていませんか？」:**
```
□ 名刺の注文ロット数が年々減っている
□ 「ネット印刷でいいや」と離れていく顧客がいる
□ 若い担当者から「名刺、要りますか？」と聞かれた
□ 新規の法人営業をしても、価格でしか勝負できない
□ このまま10年、同じ商売を続けられるか不安がある
□ 何か新しいことを始めたいが、何から手をつければいいかわからない
```
→ 3つ以上当てはまった方へ、というブリッジで次セクションへ

**エモーショナルストーリー:**
> ある地方の名刺印刷会社。創業40年、地元では知らない人はいない老舗だった。
>
> 社長の田中さん（仮名・62歳）のもとに、20年来の取引先から電話が入った。
> 「田中さん、長い間お世話になりました。来期から名刺はネット印刷に切り替えることになりまして…」
>
> 受話器を置いた田中さんは、しばらく動けなかった。
> その取引先だけで、年間30万円の売上があった。
>
> 翌週、もう1社から同じ連絡が入った。
>
> 「価格じゃ勝てない。でも、うちには40年の信頼がある。
> 　この信頼を活かせる"何か"はないだろうか——」
>
> 田中さんが見つけた答えは、意外にもシンプルなものだった。

#### ビジュアル要素
- 右肩下がりの市場グラフ（名刺印刷市場）と右肩上がりのグラフ（デジタル名刺市場）を対比表示
- チェックリストはインタラクティブに（チェックを入れられる演出。実際の送信はしない）
- ストーリー部分は背景を変え、書籍のような読み心地で

### セクション3：共感・寄り添い（Affinity）

**目的:** 「この会社は印刷業界のことをわかっている」と感じさせ、心理的安全性を確保する
**感情:** 安堵・信頼
**心理トリガー:** なし（売り込みを一切しないセクション）

#### 見出しコピー案

| # | コピー案 |
|---|---------|
| 1 | **紙の名刺の価値を、私たちは知っています。** |
| 2 | **「紙をやめて下さい」なんて、言いません。** |
| 3 | **30年、名刺を作り続けてきた御社だからこそ。** |

**推奨: 案2（明確な立場表明で安心感を与える）**

#### 本文の方向性

**紙の名刺の価値を肯定するメッセージ:**
> 手渡しの名刺交換には、デジタルでは代替できない価値がある。
> 相手の目を見て、名前を伝える。その瞬間の信頼感は、紙の名刺だけのもの。
>
> 私たちが提案するのは、「紙の代わり」ではありません。
> **「紙の名刺に、デジタルを"足す"」**という考え方です。
>
> 名刺を印刷する。そこにNFCシールを添える。
> 紙の名刺を受け取った相手が、あとからスマホでも情報を見られる。
> 「紙＋デジタル」——これが、御社の新しい提案になります。

**キーメッセージ:**
- 「デジタルが紙を置き換える」のではなく「紙を強化する」
- 名刺を作り続けてきた御社の信頼と経験がベースになる
- ITの難しいことは、すべて私たちが引き受ける

#### ビジュアル要素
- 紙の名刺とNFCシールが共存しているイメージ写真
- 温かみのあるトーン（暖色系のアクセント）
- テキスト中心のレイアウト（セールス感を排除）

### セクション4：業界の変化（Educate）

**目的:** 客観的データで「今動くべき理由」を論理的に示す
**感情:** 焦り → 希望への転換
**心理トリガー:** 権威性（市場データの引用）、バンドワゴン効果

#### 見出しコピー案

| # | コピー案 |
|---|---------|
| 1 | **名刺業界に、静かな地殻変動が起きています。** |
| 2 | **知っていましたか？ デジタル名刺市場は年6.8%で成長中です。** |
| 3 | **変化はすでに始まっている。問題は、あなたが乗るかどうかだけ。** |

**推奨: 案1（煽りすぎず、知的好奇心を刺激）**

#### 本文の方向性

**提示すべきデータポイント:**

1. **市場成長データ**
   - デジタル名刺の世界市場: 2024年 約4.2億ドル → 2032年 6.7億ドル（CAGR 6.8%）
   - 出典: Fortune Business Insights

2. **NFC普及データ**
   - iPhone: iOS 13以降（2019年〜）NFC読み取り標準対応。日本のiPhoneシェア約65%
   - Android: NFC対応率95%以上（2023年以降の新機種）

3. **行動変容データ**
   - リモートワーク定着によるオンライン名刺交換の増加
   - Z世代・ミレニアル世代の「紙の名刺を持たない」傾向

4. **競合の動き**
   - 一部の先進的な印刷会社がデジタル名刺の取り扱いを開始
   - 「すでに動いている会社がある」ことを匂わせる（具体名は出さない）

**メッセージの転換ポイント:**
> この変化は「脅威」ではなく「チャンス」です。
> なぜなら——名刺のことを一番よく知っているのは、名刺屋だから。
> IT企業にはない「顧客との信頼関係」を持つ御社だからこそ、
> デジタル名刺を提案する最適なポジションにいるのです。

#### ビジュアル要素
- 市場成長のインフォグラフィック（カウントアップアニメーション）
- タイムライン表示：2020 → 2026の変化
- アイコン付きの統計カード3〜4枚

### セクション5：解決策の提示（Solution）

**目的:** Shareのホワイトラベルモデルを「自社の新事業」として理解させる
**感情:** 期待・ワクワク
**心理トリガー:** フレーミング効果（「代理店」ではなく「自社事業」）

#### 見出しコピー案

| # | コピー案 |
|---|---------|
